# rote

Record once, replay forever.

Automate repetitive web data entry by watching a human do it once.
Copy a rectangle of data from your spreadsheet, run `rote`, fill out the form once while rote watches, and it replays the pattern for every remaining row — no scripting, no configuration, no browser extensions.

## Demo form for manual testing

A self-contained demo form lives in `examples/demo-form/`.
Open `examples/demo-form/index.html` directly in Chrome or Edge — no local server required.

It includes:
- multiple text inputs
- a select
- a textarea
- a submit button
- a simple confirmation page at `submitted.html`
